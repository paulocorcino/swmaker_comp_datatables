# Componente Webrun DataTable (Softwell Maker) — Instalação

Este repositório contém os arquivos necessários para instalar o componente **Webrun DataTable** no **Softwell Maker**.

## Instalação

1. Faça o download do conteúdo do repositório em formato **.zip** pelo GitHub.
2. Após o download do arquivo `swmaker_comp_datatables-main.zip`, **descompacte** o arquivo.
3. Acesse a pasta `Maker 5`, copie a pasta `Resources` e cole na **pasta raiz do Maker**, onde está o aplicativo `FMaker.exe`.
4. Copie as pastas `assets` e `autodeploy` e cole na pasta `Maker.Commons`, que deve existir na pasta `system` do **Webrun**.  
   - Caso a pasta não exista, **crie** a pasta `Maker.Commons` e cole as duas pastas (`assets` e `autodeploy`) **dentro dela**.
5. Feche e abra novamente o **Softwell Maker** e reinicie o **Webrun**.
6. O novo componente estará disponível na categoria **"Adicionais"** na **Paleta de Componentes**.
